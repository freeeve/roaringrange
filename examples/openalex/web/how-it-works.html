OpenAlex — static in-browser search · powered by roaringrange

This demo searches ~9.5M OpenAlex works entirely in your browser, with no backend. The index is one static file on S3, and a query fetches just a few small byte-ranges over HTTP — never the whole file. It's powered by the roaringrange library.
